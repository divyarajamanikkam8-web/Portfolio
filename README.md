# 🌐 Personal Portfolio Website

Welcome to my personal portfolio! This is a clean, responsive, and modern website showcasing my skills, projects, education, and expertise.

## 🚀 Quick Start

### Features

- 📱 **Responsive Design** - Mobile-friendly and adaptive to all screen sizes
- 🎨 **Modern UI** - Beautiful gradient styling and clean layout
- 🌙 **Dark Mode** - Toggle between light and dark themes
- ⚡ **Interactive Elements** - Smooth animations and JavaScript interactions
- 📡 **GitHub API Integration** - Dynamically display your latest projects
- 📬 **Contact Form** - Fully validated email contact form
- 🧠 **Skills Section** - Animated skill cards with visual hierarchy

## 📖 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/divyarajamanikkam8-web/Portfolio.git
   cd Portfolio
   ```

2. **Open in your browser**
   - Simply open `index.html` in your preferred browser
   - Or use a live server (VS Code extension recommended)

3. **Customize for yourself**
   - Update `index.html` with your information
   - Modify `style.css` to match your brand
   - Update the GitHub username in `script.js` for API integration
